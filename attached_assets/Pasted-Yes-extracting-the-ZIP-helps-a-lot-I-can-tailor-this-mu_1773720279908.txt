# What “enhance before resizing” means

You asked what that really means.

## Correct order

**raw image → cleanup → enhancement → upscale if needed → save master → crop → resize → export**

### Definitions

## Cleanup

Remove screenshot evidence or junk before anything else:

* UI badges
* counters
* overlay icons
* circles in corners
* app/browser edges
* borders
* text snippets

## Enhance

Improve image quality overall:

* tone
* clarity
* contrast
* recover detail carefully

## Denoise

Reduce noise/grain/compression roughness.

## Deartifact

Remove screenshot/JPEG damage like:

* blockiness
* smearing
* halos
* ugly compression texture

## Upscale

Increase image dimensions after cleanup.

## Master

A high-quality cleaned version of the image that becomes your source for all later crops.

## Crop

Choose composition for the needed aspect ratio.

## Resize

Set the exact final pixel dimensions for the site slot.

## Export

Save the final web-ready file.

---

# Why this order matters

If you resize or crop too early:

* defects get baked into the file
* hair detail gets softer
* sharpening looks fake faster
* you lose flexibility for multiple outputs

Because your portfolio is **beauty/hair/editorial**, detail around:

* hairline
* texture
* silhouette
* edge definition
  matters a lot more than with a generic portfolio.
