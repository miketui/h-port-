3. My exact recommendation for where enhancement happens

This is the order:

raw source → cleanup → enhancement → upscale if needed → master save → crop to aspect ratio → final resize → export

That is the correct sequence for your case.

What each step means

Cleanup
Remove screenshot evidence first:
	•	UI badges
	•	carousel counters
	•	floating circles/icons
	•	browser/app chrome
	•	visible borders

This matters because several of the repo images visibly still carry screenshot-style overlay artifacts. For example, the Rihanna red-carpet image includes a 1/4 badge, and other source images show floating circular UI elements in the top-right.  ￼

Enhancement
Improve the source visually:
	•	tonal correction
	•	contrast recovery
	•	mild texture recovery
	•	sharpening control

Denoise
Remove digital noise, grain, and rough compression texture.

Deartifact
Remove JPEG blockiness, halos, smearing, and screenshot compression damage.

Upscale
Increase pixel dimensions only after cleanup so you do not enlarge defects.

Master save
Create one high-quality “clean master” per image.
This is the file you derive all exports from.

Crop
Choose the framing for the required aspect ratio.

Resize
Set the final exact pixel dimensions for the site slot.

Export
Compress/save for web delivery.

4. Why enhancement should happen before resizing

Because resizing a weak screenshot first bakes in the damage.

If you crop/export too early:
	•	artifacts get harder to remove
	•	hair texture gets mushier
	•	edges get harsher
	•	your final files look processed instead of premium

For a hairstylist portfolio, this is especially important because viewers will notice bad hair detail faster than almost anything else.

5. Folder-safe production system

You said you do not want categories mixed. I agree.

Use this inside each category:

beauty/
  raw/
  masters/
  exports/
    3x4/
    4x5/
    featured/

Repeat that same pattern for every category.

Example:

public/images/portfolio/beauty/
  raw/
  masters/
  exports/3x4/
  exports/4x5/
  exports/featured/

public/images/portfolio/red-carpet/
  raw/
  masters/
  exports/3x4/
  exports/featured/

That keeps every Beauty file with Beauty, every ECOM file with ECOM, and so on.
